# Netflix Clone 

## Technologies used:
+ React
+ Node.js
+ Express
+ MongoDB
+ Redux Toolkit
+ Firebase 
+ Axios


## Steps to Start the App

+ Install React Dependencies using the following command
    npm install
+ Instal Node Dependencies using the following command
    npm install
+ Start Node App
    npm start
+ Start React App
    npm start
